# Stubborn – Boutique de Sweatshirts

---

## Installation

Prérequis

- PHP 8.2+
- Composer
- MySQL/MariaDB

- Cloner le projet
git clone
cd 
composer install

---

## Importer la base de données 

Le fichier SQL est à la racine du projet : bdd_s02.sql

Créer une base de données et importer le fichier via phpMyAdmin
